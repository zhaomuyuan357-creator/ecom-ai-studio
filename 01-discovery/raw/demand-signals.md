  - AI商品图片生成

  - AI选品工具
  - 智能客服AI
  - 主图制作 / 详情页制作
  - 淘宝店铺装修工具
  - 电商作图软件

- **双11（11月）**和**618（6月）**大促前1-2个月为搜索高峰，卖家集中备图

- 日常需求稳定，大促期间需求量可翻2-3倍

### 关键信号

- 各大电商平台（淘宝、京东、拼多多、抖音）均在加大AI工具部署力度

## 购买行为

### 如何发现工具
1. **同行推荐/卖家社群**（微信群、QQ群）— 最重要渠道
2. **平台官方推荐**（淘宝服务市场、抖音服务市场）
3. **短视频/直播教程**（抖音、B站搜"电商作图"）
4. **搜索引擎**（百度搜"电商作图工具"、"AI商品图"）
5. **电商培训课程**附带推荐

### 决策参与者
- **个体卖家**：自己决定，配偶可能参与（财务角度）
- **小团队**：运营主管决定，老板审批预算
- **中型店铺**：运营提需求 → 店长/老板审批 → 可能IT评估

### 决策标准 Top 5
1. **出图效果**：能否直接用于商品详情页，不"假"
2. **易用性**：学习成本低，10分钟上手
3. **速度**：单张图生成时间 < 2分钟
4. **价格**：性价比高，投入产出比合理
5. **批量能力**：能否一次处理多个SKU

### 典型预算
- **个体卖家**：¥300-1,000/年（工具类）；¥200-500/月（含推广工具总预算）
- **小团队**：¥1,000-5,000/年（设计工具）；¥2,000-10,000/月（总工具预算）
- **中型店铺**：¥5,000-20,000/年（设计工具）；¥10,000+/月（总工具预算）

### 销售周期
- **冲动购买**（看到效果演示后）：1-3天
- **试用后购买**：1-2周
- **比价后购买**：2-4周
- 大促前1-2个月为集中采购期

### 常见异议
1. "免费工具（万象台/即创）够用了，为什么要付费？"
2. "AI生成的图能过平台审核吗？"
3. "一次性付费后，后续更新收费吗？"
4. "效果能比得上请摄影师吗？"
5. "我不太懂AI，操作复杂吗？"
6. "能不能先试用再买？"

## 付费意愿评估

### 强WTP（付费意愿）证据

1. **电商卖家已有付费习惯**：大多数卖家已为ERP、数据分析、客服等工具付费，对SaaS工具的付费认知成熟
2. **明确的成本替代**：摄影300-5000元/套，设计师3000-12000元/月，工具价格远低于现有支出
3. **订阅疲劳**：用户对月付/年付模式产生疲劳，一次性付费（买断制）在部分用户中接受度较高，可作为差异化卖点
4. **刚需驱动**：商品图是电商运营的基础设施，不是"nice to have"
5. **平台工具的局限性**：万象台/即创虽然免费，但功能有限、模板同质化严重，有付费升级空间

### 弱WTP证据

1. **免费替代品存在**：阿里万象台、抖音即创免费，部分卖家"够用就行"
2. **价格敏感群体大**：中小卖家利润薄，对新增支出谨慎
3. **AI信任度待建立**：部分卖家对AI生成图片的质量和平台合规性存疑
4. **一次性付费 vs 订阅的接受度分化**：部分用户习惯订阅制，对一次性大额支出有心理门槛

### 推荐定价区间

| 版本 | 建议价格 | 目标客户 | 包含内容 |
|------|---------|---------|---------|
| **体验版** | 免费 | 所有人 | 每天5张，有水印 |
| **标准版** | ¥299（一次性） | 个体/小微卖家 | 无限出图，基础场景库（50+），本地运行 |
| **专业版** | ¥599（一次性） | 小团队/进阶卖家 | 标准版全部 + 高级场景库（200+）、批量处理、多平台尺寸适配 |
| **旗舰版** | ¥999（一次性） | 中型店铺 | 专业版全部 + 自定义场景训练、API接口、优先更新 |

### 理由

1. **299元定价锚点**：低于稿定设计年费（199-258元/年），但强调"一次付费永久使用"，心理价值感更高
2. **599元甜蜜点**：相当于请摄影师拍1-2套图的费用，ROI立竿见影
3. **999元天花板**：低于1000元心理关口，适合有预算审批流程的团队
4. **一次性付费差异化**：市场上绝大多数竞品为订阅制，买断制是明确的差异化优势
5. **免费版引流**：降低试用门槛，用效果说话

## 市场验证评分

| 维度 | 评分 | 说明 |
|------|------|------|
| **搜索需求** | **强** | AI商品图、电商AI工具搜索量持续上升，大促前有明显峰值 |
| **竞争活动** | **高** | 稿定设计、创客贴、美图、平台官方工具均已布局，但AI原生桌面工具仍有空白 |
| **客户支出** | **增长** | 电商卖家工具预算逐年增加，AI工具渗透率快速提升 |
| **替代品压力** | **中高** | 免费平台工具（万象台/即创）构成基线竞争，但功能和灵活性有限 |
| **痛点强度** | **强** | 作图慢、成本高、技能门槛是被反复提及的核心痛点 |
| **付费意愿** | **中强** | 已有付费习惯，但价格敏感；一次性付费模式有吸引力但需教育 |

### 总体需求信号：**强（Strong）**

**核心判断**：市场需求真实且增长中。中国电商卖家数量庞大（千万级），中小卖家对低成本、高效率的商品图生成工具有刚性需求。299-999元一次性付费定价在目标客户的预算舒适区内，且与主流订阅制竞品形成差异化。关键风险在于免费平台工具的功能提升速度，以及AI生成图片质量是否能满足平台审核要求。
